Теоретический вопрос
Обьясните своими словами, как вы понимаете, как работает прототипное наследование в Javascript

***************


В JavaScript объекты имеют специальное скрытое свойство Prototype, 
которое либо равно null, либо ссылается на другой объект. 

Когда мы хотим прочитать свойство из object, а оно отсутствует, 
JavaScript автоматически берёт его из прототипа.
В программировании такой механизм называется «прототипным наследованием».
